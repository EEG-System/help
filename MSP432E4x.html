This HELP ME page has the goal to facilitate the documentation and the usage of the files provided.

Goal

The MSP432E401Y Launchpad is the microcontroller necessary for the system as it is the middle man that will request data and manipulate the ADS1299 to pass the results to the PC.

Pre-requisites

  • Code Composer Studio

    • Empty Project Initialization.

  • USB Library



MSPs GPIO and its functionalities.
We recommend to see device datasheets for a better understading of the pins.

Why are we using this microcontroller?
The selling point using the MSP432E4x for our project was the SPI and the USB serial communication peripherals.
These peripherals allows to a simultaneous communication between the ADS1299 and the PC.
